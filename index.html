Cheatsheet HTML


HTML est le langage de balisage conçu pour représenter les pages web


Les balises


  • h1 - h6 = il y a 6 niveaux de titre
  • p = pour créer un paragraphe
  • div = conteneur utilisé pour isoler une section de la page ou un élément
  • img = pour afficher une image
  • src="url" = la source de l'image à afficher
  • ul = pour faire une liste de points
  • li = pour faire une liste
  • a href="" = pour créer des hyperliens
